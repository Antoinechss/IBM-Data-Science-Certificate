## Overview

This repository documents the successful completion of the **IBM Data Science Professional Certificate**.

The program covers the full data science pipeline, including data analysis, machine learning, data visualization, SQL, and Python-based workflows, with hands-on labs and a capstone project.

---

## Certificate

- Provider: IBM (via Coursera)  
- Scope: Data Science, Machine Learning, Python, SQL, Data Analysis  
- Format: Online professional certificate (non-credit)  

---

## Content

The certificate includes courses on:

- Data Science methodology  
- Python for data analysis and machine learning  
- SQL and databases  
- Data visualization  
- Applied machine learning  
- Capstone project
